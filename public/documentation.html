myFlix API Documentation


This API will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Return data about a director by name /movies/directors/[Name] GET None A JSON object holding data about a movie director

Example:
                        
{
    "Name": "Hayao Miyazaki",
    "Bio": "Hayao Miyazaki (宮崎 駿, Miyazaki ...",
    "Birthday": "1941-01-05T00:00:00.000Z"
}
Allow a user to register /users POST A JSON object holding data about the user to add

Example:
                        
{
    "Username": "user123",
    "Password": "pass123",
    "Email": "user123@gmail.com",
    "Birthday": "1999-09-09"
}
A JSON object holding data about the user that was added, including an ID

Example:
                        
{
    "Username": "user123",
    "Password": "pass123",
    "Email": "user123@gmail.com",
    "Birthday": "1999-09-09T00:00:00.000Z",
    "FavoriteMovies": [],
    "_id": "62683a2531a60d702f674642",
}
Allow a user to update their information by username /users/[Username] PUT A JSON object holding the changes to the users information

Example:
                        
{
    "Username": "newuser123",
    "Password": "newpass123",
    "Email": "newuser123@gmail.com",
    "Birthday": "2000-09-09"
}
A JSON object holding the users updated information

Example:
                        
{
    "_id": "62683a2531a60d702f674642",
    "Username": "newuser123",
    "Password": "newpass123",
    "Email": "newuser123@gmail.com",
    "Birthday": "2000-09-09T00:00:00.000Z",
    "FavoriteMovies": [],
}
Allow a user to add a movie to their list of favorites /users/[Username]/movies/[Title] POST None A JSON object holding the users information and updated favorite movie list

Example:
                        
{
    "_id": "62683a2531a60d702f674642",
    "Username": "newuser123",
    "Password": "newpass123",
    "Email": "newuser123@gmail.com",
    "Birthday": "2000-09-09T00:00:00.000Z",
    "FavoriteMovies": [
        "62609a97363ccb7c4449495c"
    ]
}
Allow a user to remove a movie from their list of favorites /users/[Username]/[MovieTitle] DELETE None A JSON object holding the users information and updated favorite movie list

Example:
                        
{
    "_id": "62683a2531a60d702f674642",
    "Username": "newuser123",
    "Password": "newpass123",
    "Email": "newuser123@gmail.com",
    "Birthday": "2000-09-09T00:00:00.000Z",
    "FavoriteMovies": []
}
Allow an existing user to deregister /users/[Username] DELETE None A text message indicating that a user was successfully deleted
