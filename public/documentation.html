myFlix API Documentation


This API will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Request URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None A JSON object that contains all the movies in the database
Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user /movies/[Title] GET None A JSON object holding data about a single movie
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genres/[GenreName] GET None A JSON object holding data about a genre
Return data about a director (bio, birth year, death year) by name /movies/directors/[Name] GET None A JSON object holding data about a movie director
Allow a user to register /users POST A JSON object holding data about the user to add

Example:
                        
            {
                "name": "Gabriel Jardio",
                "username": "gjardio",
                "email": "gabrieljardio@gmail.com",
                "favorite-movies": ["Interstellar", "Dunkirk"]
            }
A JSON object holding data about the user that was added, including an ID
Allow a user to update their user info (username) /users/[ID] PUT A JSON object holding the changes to the users information A text message indicating whether the users information was successfully changed
Allow a user to add a movie to their list of favorites /users/[ID]/[MovieTitle] POST None A text message indicating whether the desired movie was added to the users list of favorites
Allow a user to remove a movie from their list of favorites /users/[ID]/[MovieTitle] DELETE None A text message indicating whether the desired movie was deleted in the users list of favorites
Allow an existing user to deregister /users/[ID] DELETE None A text message indicating that a user email has been removed
